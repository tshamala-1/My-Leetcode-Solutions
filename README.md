# My LeetCode Solutions

## 📖 About
This repository contains my **LeetCode solutions** in Java.  
It includes the problem statements, explanations, approaches, and working code for each problem.  
The goal is to **practice algorithms and data structures** and provide a clear, organized reference for others.

## ✅ Features
- Each problem has a **.md file** with:
  - Problem statement
  - Intuition
  - Step-by-step approach
  - Clean, working Java code
  - Time and space complexity analysis
  - Edge cases and notes


## 🚀 How to Use
1. Clone the repository:

```bash
git clone https://github.com/tshamala-1/My-Leetcode-Solutions.git
```

Navigate to the folder of interest (e.g., HashMap):
cd My-Leetcode-Solutions/HashMap

Open the .md files to study the solution or view the code.
